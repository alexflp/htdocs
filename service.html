QR Code Gateway


1. Customers choose WeChat Pay or Alipay channel after website integrated in AlphaPay



2. Website call AlphaPay Api to create order and display the QR Code in their page or redirect to AlphaPay's payment page



3. Customers use their payment App to scan the QR Code



4. Finish payment on the page opened after scanning


JS API


1. Guide customers open website in WeChat Pay or Alipay App



2. Create order with AlphaPay API. Redirect to AlphaPay payment page.


3rd Party Gateway


1. Redirect to Alipay Payment page from PC website


2. Use Alipay App to scan QR Code and finish payment


APP Gateway


1. Open shopping app and commit order


2. Call Alipay SDK to pay


3. Finish payment


Merchant QrCode


1. Customers use their payment App to scan the merchant static QR Code


2. Customers input price and press pay button



3. Finish payment




POS


1. Merchant input price on POS and press generate QR Code button


2. Customers use their payment App to scan the QR Code displaying on the POS screen


3. Customers confirm to pay


4. Finish payment



Retail API


1. Customer show their payment QR Code in their apps and merchant use a scanner to scan the QR Code


2. Call AlphaPay api to finish payment
